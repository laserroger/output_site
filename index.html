New update tool

Here I introduced auto-run python code to check whether the website content have been modified remotely. Considering this site is only running in Local Area Network, any approach to modify it when out of this LAN is not directly possible. I then setup an auto-run program to pull from github, and I just have to upload my update to the website to github. There might exists some kind of delay due to this mechanism, but I think it won't be a problem.

Now (Mar 5, 17:14, GMT+8) it's online.
